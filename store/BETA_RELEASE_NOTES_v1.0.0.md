# ShopSphere v1.0.0 — Release Notes

## What's New

Welcome to ShopSphere — your premium shopping destination!

### Features
- Browse curated product collections with a modern, premium UI
- Featured products with category filtering
- Smart search across the entire catalog
- Secure account registration and login with email verification
- Wishlist to save your favorite products
- Shopping cart with size and color selection
- Secure checkout with delivery address
- Order history with expandable order details
- Push notifications for orders and promotions
- Profile management with edit personal details
- Admin dashboard for staff and managers (role-based)

### Improvements
- Optimized image loading with caching for smooth scrolling
- Offline-friendly product browsing with local catalog fallback
- Accessibility support for screen readers
- Free shipping on qualifying orders (configurable via Remote Config)

### Known Limitations (v1.0.0)
- Payment processing is simulated (no real payment gateway yet)
- Product catalog uses local fallback when Firestore is empty
- Admin CRUD operations require Cloud Functions deployment
